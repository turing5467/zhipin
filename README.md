# 

### React项目实战

Boss直聘仿写，实现基本功能



#### 技术栈

- 前端主要使用库：React + React-Router + Redux + axios 
- UI框架：antd
- 后端：Express + mongodb + multer + ws



#### 实现功能

1. 登录、注册
2. 职位/公司展示与筛选
3. 在线编辑简历
4. 简历/头像上传（至服务端）、简历下载
5. 聊天（其实是自动回复，双向聊天还不好实现，因为没扒招聘人的信息，所以就只做了简单的自动回复）



#### 亮点

1. 对axios进行封装，降低日后维护成本
2. 

#### 性能优化

1. 图片加载
